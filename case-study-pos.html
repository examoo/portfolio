Project Overview

AllZahrani is a robust Point of Sale (POS) and Inventory Management System designed for high-volume retail businesses. The client required a solution that could handle thousands of SKUs, manage multiple warehouses, and provide real-time financial reporting.

The system provides a seamless checkout experience for cashiers while giving administrators powerful tools to track sales trends, manage stock levels, and generate tax-ready invoices.

Key Features

  • Dual-Interface POS: Fast terminal for cashiers and detailed backend for managers.
  • Multi-Warehouse: Transfer stock between locations and track inventory movement.
  • Financial Reporting: Automated P&L statements, expense tracking, and tax reports.
  • Barcode Integration: Support for barcode scanners and thermal printers.

Tech Stack

  • Backend Laravel
  • Frontend Vue.js
  • Database MySQL
  • Real-time Pusher
  • Reporting Chart.js
