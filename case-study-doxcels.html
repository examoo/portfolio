Project Overview

Doxcels is a modern, cloud-based Hospital Management System (HMS) designed to streamline operations for clinics and hospitals. The goal was to build a scalable, multi-tenant platform that could serve dozens of independent healthcare facilities simultaneously while ensuring data isolation and security.

The system moves beyond traditional paper-based records to a fully digital workflow, covering patient registration, OPD (Outpatient Department), IPD (Inpatient Department), laboratory management, and financial billing.

Technical Architecture

We architected Doxcels as a multi-tenant SaaS application where a single code base serves multiple clients (tenants). Each tenant's data is logically isolated within a shared PostgreSQL database using Row Level Security (RLS) and scope bindings, ensuring strict privacy.

The frontend utilizes Vue.js for a reactive, single-page application (SPA) experience, communicating with a robust Laravel API.

Key Features

  • Smart OPD Management: Efficient patient queuing, token generation, and doctor appointment scheduling.
  • Laboratory Module: Integrated lab test bookings, result entry, and automated report generation with QR codes.
  • IPD & Ward Management: managing bed occupancy, admissions, and discharges.
  • Financial Dashboard: Real-time tracking of daily revenue, expenses, and doctor shares.
  • WhatsApp Integration: Automated patient notifications for appointments and reports.
